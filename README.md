# contenedores: filas y columnas

## actividad: 

1. Crear cuatro columnas en bootstrap. Aplicar diferentes colores

2. Crear dos columnas y dos filas. Usar diferentes colores para cada una de ellas.

3. Crear tres filas, laprimera contendra dos columnas, la segunda tres, y la tercera dos.


### 1. Cuatro columnas

container: para centrar el contenido.

row: para crear una fila.

col: para dividir automáticamente en 4 columnas iguales.

#### Clases de color:

bg-primary, bg-success, bg-danger, bg-warning

text-white / text-dark: para que el texto se vea bien.

p-3: padding interno.



### 2. Dos columnas y dos filas

Dos row → crean dos filas

En cada fila:

Dos col → crean 2 columnas

Colores diferentes en cada columna para distinguirlas.

  col
  
  col

  
### 3. Tres filas (2, 3 y 2 columnas)

3 row → tres filas

Distribución:

Fila 1 → 2 col

Fila 2 → 3 col

Fila 3 → 2 col

Cada columna con colores distintos.



container → contenedor principal

row → fila

col → columna (se adapta automáticamente)

bg-* → colores

p-3 → espacio interno
